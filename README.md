\# 🧠 Memory Game (Jogo da Memória)



A web-based memory game developed using \*\*HTML, CSS, and JavaScript\*\*.



\## 🎮 Features

\- Card flip animation

\- Matching logic

\- Attempt counter

\- Victory message



\## 🚀 Technologies

\- HTML5

\- CSS3

\- JavaScript
